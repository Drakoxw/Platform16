# Platform16

### Run your Storybook
```sh
    ng run Platform16:storybook
```

### Generate Module
```sh
    ng g module [nameModule]
```

### Generate Component -Module
```sh
    ng g c [nameModule]/[nameComponent] --module=client 
```
>Example -> `ng g c shared/button --module=shared --style none`
* --skip-tests
* --style none
